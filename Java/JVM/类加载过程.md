#### 类的生命周期

---

类从被加载到虚拟机内存中到卸载出内存, 它的整个生命周期可以简单概括为7个阶段:

1. 加载(loading)
2. 验证(verification) - 链接(linking)
3. 准备(preparation) - 链接
4. 解析(resolution) - 链接
5. 初始化(initialization)
6. 使用(using)
7. 卸载(unloading)
