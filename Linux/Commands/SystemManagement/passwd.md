Linux passwd命令用来更改使用者的密码.

**语法:**

```
passwd [-k] [-l] [-u [-f]] [-d] [-S] [username]
```

**参数说明:**

- -d 删除密码
- -f 强迫用户下次登录时必须修改口令
- -w 口令要到期提前警告的天数
- -k 更新只能发送在过期之后
- -l 停止账号使用
- -S 显示密码信息
- -u 启用已被停止的账户
- -x 指定口令最长存活期
- -g 修改群组密码
- 指定口令最短存活期
- -i 口令过期后多少天停用账户

**实例:**

修改用户密码:

```
passwd runoob
```

显示账号密码信息:

```
passwd -S runoob
```

删除用户密码:

```
passwd -d runoob
```
